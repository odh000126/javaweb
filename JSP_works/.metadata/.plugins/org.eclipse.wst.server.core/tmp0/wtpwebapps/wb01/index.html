쇼핑몰회원관리 프로그램
쇼핑몰 회원정보와 회원매출정보 데이터베이스를 구축하고 회원관리 프로그램을 작성하는 프로그램이다.
프로그램 작성 순서
  • 회원정보 테이블을 생성한다.
  • 회원정보, 매출정보 테이블에 제시된 문제지의 참조데이터를 추가 생성한다.
  • 회원정보 입력 하면프로그램을 작성한다.
  • 회원정보 조회 프로그램을 작성한다.
  • 회원매출정보 조회 프로그램을 작성한다.
